# 🚀 Smart AI Task Agent with LLM Integration

## 📌 About

Smart AI Task Agent is a MERN-based web application integrated with a Large Language Model (LLM) to intelligently process and summarize user inputs.
The project bridges the gap between traditional task systems and AI by enabling users to input data and receive concise, meaningful summaries in real-time.

This system demonstrates how modern AI (LLMs) can be seamlessly integrated into full-stack applications to enhance productivity and automation.

---

## ✨ Features

* 🔹 AI-powered text summarization
* 🔹 Clean and interactive UI built with React
* 🔹 Real-time input → output processing
* 🔹 Modular component-based frontend architecture
* 🔹 Backend API handling LLM requests
* 🔹 Scalable structure for future AI features
* 🔹 History tracking of inputs and outputs

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js

### AI Integration

* OpenAI API (LLM)

### Database (Optional / Future Scope)

* MongoDB

---

## ⚙️ Workflow

1. User enters input text in the frontend
2. Request is sent to backend API
3. Backend processes request and sends it to LLM
4. LLM generates summarized output
5. Backend returns response to frontend
6. Frontend displays summarized result
7. Input-output history is maintained

## 📌 How to Run

### 1. Clone the repository

```
git clone https://github.com/your-username/Smart-AI-Task-Agent-with-LLM-Integration.git
```

### 2. Setup Backend

```
cd Backend
npm install
npm start
```

### 3. Setup Frontend

```
cd Frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```
OPENAI_API_KEY=your_api_key_here
```
